# Security Policy

## Supported Versions

Security fixes are currently provided for the latest published version.

| Version | Supported |
| ------- | --------- |
| 1.x     | Yes       |

## Reporting a Vulnerability

Please report suspected security issues privately by opening a GitHub security advisory for this repository.

If a GitHub advisory is not practical, contact the maintainer through the GitHub profile linked from this repository. Please include:

- The affected package and version.
- A concise description of the issue.
- Steps to reproduce or validate the finding.
- Any known impact or workaround.

Please do not open public issues for vulnerabilities until a fix or mitigation is available.
